vgteam

vgteam is a collaboration working on tools for graph-based genomics.

Choose from any of the fine vgteam products below:

vg

vg, the variation graph toolkit, allows for constructing, manipulating, and analyzing genome graphs.

toil-vg

toil-vg is a wrappr around vg which uses UCSC's Toil workflow manager to parallelize vg analyses in the cloud.

HAMP

HAMP contains tools for evaluating vg's haplotype-aware mapping capabilities
